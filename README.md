# Pipways - Institutional Forex Trader Development Platform

A comprehensive trading education platform with AI-powered chart analysis, trade journaling, course management, and mentorship.

## 🚀 Features

- **AI Chart Analysis** - Upload charts for instant professional-grade analysis
- **Trade Journal** - Comprehensive logging with analytics
- **Course Management** - Structured learning paths with modules
- **AI Mentor** - 24/7 trading psychology and strategy guidance
- **Payment Integration** - Paystack for Nigerian market
- **Telegram Bot** - Lead capture and support automation
- **Blog System** - Content marketing with SEO-friendly URLs
- **Admin Dashboard** - User management, analytics, and revenue tracking

## 🛠️ Tech Stack

- **Backend**: FastAPI, SQLAlchemy, PostgreSQL
- **Frontend**: Jinja2 Templates, Tailwind CSS, Vanilla JS
- **AI**: OpenRouter (Claude 3.5 Sonnet)
- **Payments**: Paystack
- **Bot**: Telegram Bot API
- **Hosting**: Render
