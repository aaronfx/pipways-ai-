## ✨ Features

### Current (Static Version)
- 📊 **Dashboard** - Real-time trading metrics and performance stats
- 📷 **Trade Journal** - Log trades manually with pair, direction, pips, and grade
- 🛡️ **Discipline Score** - Track your trading discipline metrics
- 🤖 **AI Mentor** - Chat interface for trading psychology coaching
- 💾 **Local Storage** - All data saved in browser (private, no server needed)

### Coming Soon (Full Stack)
- 🔐 User accounts & cross-device sync
- 🤖 AI screenshot analysis (OCR trade extraction)
- 📈 Advanced analytics (equity curves, session performance)
- 🔔 Real-time discipline alerts
- 💳 Subscription management
