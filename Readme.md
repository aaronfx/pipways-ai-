# Pipways - Institutional Trader Development Platform

A professional forex trading journal and AI-powered chart analysis platform built with FastAPI and modern web technologies.

![Pipways Dashboard](https://your-screenshot-url-here.png)

## 🚀 Features

- **Trade Journal**: Log and track all your trades with detailed metrics
- **AI Chart Analysis**: Upload trading charts for instant AI-powered analysis
- **Discipline Score**: Track your trading psychology and consistency
- **AI Mentor**: Get personalized trading psychology coaching
- **Performance Dashboard**: View win rates, total pips, and trade history

## 🛠️ Tech Stack

- **Backend**: FastAPI (Python)
- **Frontend**: HTML5 + Tailwind CSS + Vanilla JavaScript
- **Database**: PostgreSQL (via asyncpg)
- **AI**: OpenRouter API (Claude 3.5 Sonnet)
- **Authentication**: JWT tokens with bcrypt
- **Hosting**: Render.com

## 📋 Prerequisites

- Python 3.11+
- PostgreSQL database
- OpenRouter API key (for AI features)
